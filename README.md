# Dice Game

A simple two-player dice game built with HTML, CSS, and JavaScript.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-dicegame.mdjaveedkkhan.me-2ea44f)](https://dicegame.mdjaveedkkhan.me)

## Live Demo

Play the game here: [dicegame.mdjaveedkkhan.me](https://dicegame.mdjaveedkkhan.me)

## Features

- Random dice roll for two players
- Winner or draw result shown instantly
- Roll the dice with a button instead of refreshing the page

## How to Play

1. Open `index.html` in a browser.
2. Enter both player names when prompted.
3. Click the Roll Dice button to roll the dice and see the result.

## Project Structure

- `index.html` - Main page structure and game controls
- `styles.css` - Page styling
- `script.js` - Game logic and dice roll handling
- `images/` - Dice images used by the game

## Notes

The game now uses the Roll Dice button to generate new dice rolls.
