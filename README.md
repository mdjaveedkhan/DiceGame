# Dice Game

A simple two-player dice game built with HTML, CSS, and JavaScript.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-dicegame.mdjaveedkkhan.me-2ea44f)](https://dicegame.mdjaveedkkhan.me)

## Live Demo

Play the game here: [dicegame.mdjaveedkkhan.me](https://dicegame.mdjaveedkkhan.me)

## Features

- Random dice roll for two players
- Winner, loser, or draw result shown instantly
- Simple setup that runs in any browser

## How to Play

1. Open `index.html` in a browser.
2. Enter both player names when prompted.
3. Refresh the page to roll the dice and see the result.

## Project Structure

- `index.html` - Main page structure
- `styles.css` - Page styling
- `script.js` - Game logic
- `images/` - Dice images used by the game

## Notes

This project uses a page refresh to generate new dice rolls.
